Open Source Engine

Trick API

A highly optimized Video Frame Extractor in Rust. Blazing fast, ultra-low memory footprint, and infinitely scalable.

Zero OOM Errors

Incoming payloads are streamed as chunked multiparts directly to disk. This allows safe parsing of 500MB+ videos with barely any RAM pressure.

Built-in Quotas

Native embedded SQLite database automatically tracks API key usage. Automatically limits and rejects requests that surpass their quota limits.

terminal
# Extract exactly the frame at 25m 40s
curl -X POST https://api.yourdomain.com/extract-frame \
  -H "x-api-key: trick-a1b2c3d4..." \
  -F "video=@meeting_recording.mp4" \
  -F "minute=25" \
  -F "second=40" \
  --output target_frame.jpg
